## Features:
- Crear habitos
- Eliminar habitos
- Visualizar habitos en un calendario
- Log In/Out
- Comprobación diaria de cumplimiento de metas
- Dashboard de visualizacion de habitos
- Prediccion de fallo en cumplimiento de habitos
